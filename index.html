InstantID supports identity-preserving generation in high fidelity with only single reference face image.

Abstract

Existing personalization generation methods, such as Textual Inversion, DreamBooth, and LoRA, have made significant progress in custom image creation. However, these works require expensive computational resources and time for fine-tuning, and require multiple reference images, which limits their application in the real world. To solve these problems, we introduce InstantID, a plug-and-play module that can generate personalized images in any style using only one face image while maintaining high fidelity. Specifically, we extract face embeddings in images as additional control conditions and concatenate them with text embeddings. To further increase fidelity, we add a controllable branch to inject ID information. Our work is compatible with common pretrained text-to-image diffusion models such as SD1.5 and SDXL as a plugin. Our results show exciting potential applications in downstream tasks. Code and pre-trained checkpoints will be made public soon!
